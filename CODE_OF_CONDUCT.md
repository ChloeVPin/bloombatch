# Code of Conduct

BloomBatch is a public project. Keep discussion focused, respectful, and useful.

## Expected behavior
- Be constructive in issues, pull requests, and reviews.
- Give clear technical feedback without personal attacks.
- Respect different experience levels and working styles.
- Assume good intent until the evidence says otherwise.

## Unacceptable behavior
- Harassment, discrimination, or threatening language
- Dismissing or mocking contributors
- Repeatedly derailing technical discussion
- Publishing private information without consent

## Reporting problems
If you are uncomfortable with a conversation or behavior in this repository, report it through a private GitHub security or contact channel associated with the repository owner.

## Scope
This code of conduct applies to repository discussions, pull requests, and any project spaces linked from this repository.
